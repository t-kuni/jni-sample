# 実行環境
Ubuntu

# 実行方法
make.shを実行する

# 各リヴィジョンについて

| rev | desc |
| --- | ---- |
| master(branch) | jniが動作確認できる最小限のセット |
| pthread(branch) | jniでpthreadを使用するサンプル |
| pthread2(branch) | pthread_cleanup_pushを使用するサンプル |
| v1(tag) | pthread(branch)と同じ |
| v2(tag) | char test[X][Y] から String[] への変換サンプル |
| v3(tag) | sprintfで文字列の結合 |
| v4(tag) | pthread2と同じ |
